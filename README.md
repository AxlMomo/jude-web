# jude-web

Public web pages for the **JUDE** sticker app, served free via GitHub Pages.

| Page | URL | Purpose |
|---|---|---|
| `confirm/` | https://axlmomo.github.io/jude-web/confirm/ | Landing page for sign-up email-confirmation links |

Why this repo exists: Supabase's free shared domain refuses to render HTML
(anti-phishing rule), and the app doesn't have its own domain yet. When a real
domain exists, these pages move there and this repo can be archived.

No secrets, no user data — static HTML only.
